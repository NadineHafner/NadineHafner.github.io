There are some very important questions and facts about Web!

  1. How can the Internet be explained? The Internet is a system of connected networks, using the TCP/IP - protocoll. The messages, devided into packets can go through different ways - there is no central regulation
  2. What is the WWW? Important: WWW is no analogy for the Internet! It is only an application containing ressources (files, databases...) and links to other ressources
  3. What is the Client - Server - Model? Division of labour between Server and Client:
    • Server: Has a static IP-Adress, is always online and answers to requests of Clients
    • Client: Can have a dynamic IP-Adress, doesn't have to be online all the time and communicates with a Server
  4. What is the OSI - Model? The Open Systems Interconnection model (OSI model) is a conceptual model that characterises and standardises the communication functions of a telecommunication or computing system without regard to its underlying internal structure and technology.
  5. How can an IP - adress be defined? An IP - adress is an Internet - adress of a computer, enabling unique identification. The ISP dynamically provides IP - adresses to clients.
  6. What is URI?URIs identify ressources definitely. DNS converts a domain into its belonging IP - adress
  7. What is the scheme? Identifies the type of the ressource and the used protocoll. For example : HTTP, FTP
  8. What is the authority and what does it consist of?Instance, which administrates the names in a namespace. Consists of userinfo(username/username: password) and host(port: Nr)
  9. What is HTTP and which steps need to be done?It is an application log, defining, how communication between Client and Server works.
    1. connection establishment through TCP/IP
    2. request of information by Client
    3. answer of the Server
    4. connection termination
